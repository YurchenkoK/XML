# Web Technologies — Lab Work

> 4th semester coursework, BMSTU

A collection of lab assignments and homework covering JavaScript, TypeScript, and web component development.

## Contents

| Directory | Description |
|---|---|
| `Homework/` | JavaScript exercises (array manipulation, async I/O) |
| `lab1_2/` | Interactive HTML calculator with custom styling |
| `lab3/` | Product catalog SPA built with vanilla JS web components and Bootstrap 5 |
| `lab4/` | Extended components |
| `lab5/` | TypeScript and build tooling |
| `lab6/` | Advanced topics |

## Tech Stack

- JavaScript (ES6+), TypeScript
- HTML5, CSS3
- Bootstrap 5
- Node.js (for component-based labs)

## Running

Labs with a `package.json` can be started with:

```bash
npm install
node index.js   # or open index.html directly in a browser
```
